1.	需要注意的是，ucp2p 库依赖于其他库，请确认其它库已经安装好
	依赖的库有：
			-lpthread -lssl -lcrypto -lm -ljson
	如果没有安装好对应的lib库，请执行：
	apt-get install openssl-dev
	apt-get install libjson0


2.	编译 ucp2p 库。
	具体编译步骤参考 ucp2p 库中的 ReadMe.txt 文件
	
3.	把 .so 文件拷贝到目标文件系统的  /usr/lib/ 目录下

4.	然后把 ucp2p 库的 include 文件拷贝到当前目录下

5.	执行 make 既可以编译出 p2p_client
